# cat

## Purpose

`cat` displays the contents of a text file directly in the terminal.

## Syntax

```bash
cat filename
```

## Example

```bash
cat readme.txt
```

Displays the content of `readme.txt`.

## Common Uses

- Read text files
- Combine multiple files
- Create simple text files

## Example

```bash
cat file1.txt file2.txt
```

Displays both files one after another.
