# sort

## Purpose

`sort` arranges lines of text in alphabetical or numerical order.

## Example

```bash
sort names.txt
```

Sorts the contents alphabetically.

## Common Options

| Option |    Description                  |
|--------|---------------------------------|
|  `-n`  | Numeric sort                    |
|  `-r`  | Reverse order                   |
|  `-u`  | Remove duplicates while sorting |

## Use Cases

- Organize data
- Prepare data for `uniq`
