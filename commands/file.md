# file

## Purpose

`file` identifies the type of a file instead of displaying its contents.

## Syntax

```bash
file filename
```

## Example

```bash
file myfile
```

Possible output:

```
myfile: ASCII text
```

## Why It Is Useful

It helps determine whether a file is:

- Text
- Binary
- Image
- Archive
- Executable

## Use Cases

- Identify unknown files
- Find human-readable files
- Analyze downloaded files
