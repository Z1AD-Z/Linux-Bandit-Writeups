# ls

## Purpose

`ls` lists the files and directories inside the current directory.

## Syntax

```bash
ls [options]
```

## Examples

```bash
ls
```

Lists the current directory.

```bash
ls -l
```

Displays files in a detailed format.

```bash
ls -a
```

Shows hidden files.

```bash
ls -la
```

Shows all files with detailed information.

## Common Options

| Option  |     Description           |
|---------|---------------------------|
|   `-l`  | Long listing format       |
|   `-a`  | Show hidden files         |
|   `-h`  | Human-readable file sizes |

## Use Cases

- View files
- Check permissions
- Find hidden files
