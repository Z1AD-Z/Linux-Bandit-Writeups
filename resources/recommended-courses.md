# Recommended Courses

- Linux Foundation's "Introduction to Linux" (LFS101) on edX — free,  official, widely respected. Covers everything from filesystem structure to basic admin tasks. Great starting point if you want a structured, credible intro.

- "Linux Journey" (linuxjourney.com) — free, self-paced, well-organized into bite-sized modules (grasshopper level through sysadmin topics). Very beginner-friendly with a clean interface.

- OverTheWire: Bandit (overthewire.org/wargames/bandit) — free, hands-on wargame that teaches command-line skills through practical challenges. Excellent if you learn by doing rather than reading/watching.

- "Linux Server Administration" or similar on Udemy (e.g., courses by Jason Cannon or Imran Afzal) — usually low-cost (often on sale for $10-15), practical, video-based, good for sysadmin-track learners who want a guided project-based approach.

- KodeKloud's Linux courses — mix of free and paid content, strong hands-on labs with real terminals in-browser. Particularly good if you're heading toward DevOps, since they also cover Docker/Kubernetes with Linux as the foundation.
