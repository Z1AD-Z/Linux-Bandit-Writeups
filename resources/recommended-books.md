# Recommended Books

- The Linux Command Line by William Shotts — free PDF at linuxcommand.org. Best starting point; covers shell basics through scripting.
- Advanced Linux Programming by Mark Mitchell, Jeffrey Oldham, Alex Samuel — full book released free under open license. Great once you're past basics and want processes, threads, IPC.
- Linux Kernel documentation (kernel.org) — official, actively maintained, free. The way to go if you want to understand internals without a paywall.
- Arch Wiki — not a "book" in the traditional sense, but free, exhaustive, and arguably the single best practical Linux reference that exists, distro-agnostic.
